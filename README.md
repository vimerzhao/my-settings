# oh-my-config
整理了的我的常用配置，方便以后迁移环境（换电脑、重装系统等）时能够快速配置。
平台相关的配置放在了系统对应目录(`Windows`和`Linux`)，其他目录的要么和系统无关（如`pic-res`），要么已经做了兼容（如`spf13-vim-local`）。

# introduction

## Windows
Window平台相关的配置：
- `xmodmap.reg`:更改键盘映射
- `VisualStudio`:VS配置，好久没用，估计过期了。
